<p align="center"><img src="https://socialify.git.ci/Shyam3000/AngularAuthRoleManager/image?description=1&amp;font=Rokkitt&amp;name=1&amp;theme=Light" alt="project-image"></p>

<p id="description">AngularAuthRoleManager is a secure web application built on Angular emphasizing role-based access control. It facilitates user registration login and role-specific feature access. Administrators can manage accounts activate/deactivate and modify roles. The application integrates seamlessly with JSON Server for efficient data handling. Featuring an intuitive UI it simplifies user authentication and access control. With robust security measures including password hashing and HTTPS AngularAuthRoleManager ensures data integrity offering a flexible and scalable solution for diverse applications.</p>

<h2>üßê Features</h2>

Here're some of the project's best features:

*   User Authentication: Enable secure registration and login functionality for users to access the application
*   Role-based Access Control: Implement role-based access control (RBAC) to manage permissions and restrict access to certain features based on user roles.
*   User Management: Allow administrators to manage user accounts including activation deactivation and role assignment.
*   Dashboard Interface: Provide a user-friendly dashboard interface for users to view relevant information or perform actions based on their role.
*   Security Measures: Ensure data security with password encryption HTTPS encryption and input validation to protect user information and prevent unauthorized access

<h2>üõ†Ô∏è Installation Steps:</h2>

<p>1. Clone the Repository</p>

```
git clone https://github.com/Shyam3000/AngularAuthRoleManager.git
```

<p>2. Navigate to the Project Directory</p>

```
cd AngularAuthRoleManager
```

<p>3. Install Dependencies</p>

```
npm install
```

<p>4. Start the JSON Server</p>

```
json-server --watch db.json
```

<p>5. Start the Angular Development Server</p>

```
ng serve
```
